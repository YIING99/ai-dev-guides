# 仓库结构与分类规则

> ⚠️ 请把下面的路径和分类修改为你自己的仓库

## 仓库路径

本地：`/Users/你的用户名/你的仓库名/`
远程：`git@github.com:你的GitHub用户名/你的仓库名.git`（SSH）

## 分类目录

| 目录前缀 | 适用内容 |
|----------|---------|
| `tutorials/` | 教程类文章 |
| `guides/` | 指南类文章 |
| `notes/` | 学习笔记 |

> 按你自己的仓库目录结构修改上面的分类。

## 目录命名

用英文短横线连接，反映核心主题，例如：
- `getting-started` — 入门教程
- `feishu-setup` — 飞书接入

## 已有教程（避免重复）

> 在这里列出你仓库中已有的教程，每次新增后更新此列表。

- `tutorials/example/` — 示例教程

## README.md 索引格式

每行格式：
```
| [中文标题](目录路径/) | 一句话简介 | 入门/中级/高级 |
```
