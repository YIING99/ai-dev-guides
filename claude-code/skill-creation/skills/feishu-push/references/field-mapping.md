# 飞书多维表格字段映射

> 根据你自己的飞书多维表格字段来修改这个文件。
> 下面是一个示例配置，你需要按照自己创建的表格字段来调整。

## 资讯库（示例）

```json
{
  "标题": "15字以内",
  "内容类型": "📰 文章摘录 | 💡 灵感闪现 | 🔗 工具发现 | 🧠 认知升级 | 📝 方法论 | 🗣️ 对话精华",
  "来源平台": "公众号 | 推特 | Reddit | Hacker News | 个人思考 | AI对话 | 其他",
  "原文链接": {"link": "url", "text": "[来源·作者] 主题"},
  "原始输入": "对话中的原始内容摘要",
  "核心洞察": "📌 技术事实：...\n💡 启发提炼：...\n🔥 个人原创：...",
  "我的思考": "你的个人视角",
  "行动项": "可实践的下一步",
  "标签": ["AI工具","自动化","开发技巧","行业趋势","效率方法论","其他"],
  "关联技术栈": ["Claude Code","飞书","其他"]
}
```

## 开发知识库（示例）

```json
{
  "文本": "20字以内标题",
  "类型": "技术实施 | 踩坑记录 | 自动化 | 功能增强 | 知识点 | 待办事项",

  "问题背景": "背景上下文",

```

## 自定义提炼视角

提炼时联想你自己的背景和产品，例如：
- 你的职业背景 + 技术学习方向 = 你的跨界视角
- 你的产品和项目列表
- 你的内容受众画像
- 你的核心认知
